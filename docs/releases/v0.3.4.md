## inD3X Art v0.3.4

**Performance, security & reliability release** — catalog icon pipeline, Rust concurrency, path safety, and cross-platform CI.

### Downloads

| Platform | Artifact |
|----------|----------|
| **Windows** | `inD3X Art_*_x64-setup.exe` (NSIS installer) |
| **Linux** | `inD3X Art_*_amd64.AppImage` (portable) |

SHA256 checksums are attached as `SHA256SUMS` on this release.

### Highlights

- **Catalog icons** — key-scoped subscriptions, bake generation tokens, stable virtual grid scroll.
- **3D viewer** — per-consumer animation textures, serialized paint strokes, UV hit fixes, extrusion cap 128².
- **Rust backend** — path safety (symlinks/junctions), incremental index, async saves, cache v2, watcher debounce.
- **CI** — `cargo test` + Clippy on Linux and macOS; Codecov upload no longer blocks merges.

### Upgrade notes

- App cache moves to **v2** under the user app data directory. Reopen projects once after upgrade.
- Settings are compatible with v0.3.2.
