## inD3X Art v0.3.6

**Premium UI & studio polish** — design tokens, glass shell, catalog/viewer/editor visual upgrade, UV rotation parity, 3D paint locking, and catalog stability fixes.

### Downloads

| Platform | Artifact |
|----------|----------|
| **Windows** | `inD3X Art_*_x64-setup.exe` (NSIS installer) |
| **Linux** | `inD3X Art_*_amd64.AppImage` (portable) |

SHA256 checksums are attached as `SHA256SUMS` on this release.

### Highlights

- **Premium UI** — tokens, glass panels, motion, tooltips, modals, toasts, command palette, and theme-aware surfaces (dark / light / high-contrast).
- **Catalog** — skeleton grid, premium empty states, keyboard roving tabindex, stable icon cache snapshots, virtual grid perf.
- **Block Studio & 3D** — floating viewport controls, compare viewport polish, UV rotation sync 2D↔3D, face-locked 3D paint, mini-scene texture sync.
- **Reliability** — session restore race fix, throttled texture reload during strokes, backend catalog/asset IPC hardening.
- **Cleanup** — removed dead toolbar CSS and unused `CatalogGridToolbar`.

### Upgrade notes

- Compatible with v0.3.5 settings and cache layout.
- Reopen projects if catalog icons or labels look stale after upgrade.
