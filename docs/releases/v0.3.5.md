## inD3X Art v0.3.5

**Block Studio & paint engine release** — creative catalog parity, studio face workflow, lazy pixel cache, abortable IPC, and catalog/viewer performance hardening.

### Downloads

| Platform | Artifact |
|----------|----------|
| **Windows** | `inD3X Art_*_x64-setup.exe` (NSIS installer) |
| **Linux** | `inD3X Art_*_amd64.AppImage` (portable) |

SHA256 checksums are attached as `SHA256SUMS` on this release.

### Highlights

- **Block Studio** — vanilla lang/creative tabs, 3D face chrome, mini-scene preview, compare-before snapshot, tier-1 icon batch IPC.
- **Paint engine** — lazy layer reads, regional composite on strokes, immutable document store, stroke batching, face UV transfer.
- **Reliability** — abortable catalog IPC (Rust + frontend), studio parallel resolve, project cache invalidation dedup, reindex logging.
- **Performance** — texture/material cache budgets, global animation loop, catalog icon pipeline prefetch, parallel CI + regression suite.
- **Quality** — shared fuzzy golden vectors (TS ↔ Rust parity).

### Upgrade notes

- Compatible with v0.3.4 settings and cache v2.
- Switch catalog language in Studio toolbar if block names should match a specific pack locale.
